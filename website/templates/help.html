Help

  • To view detailed information on a game, click on its name in the collection page. Alternatively, you can also type the name of the game in the search box and choose the game from the resulting suggestions.
  • To add a game, click the add button on the navigation bar at the top. Information about a Steam game can be pre-filled with the Steam app ID. Using the Steam app ID will take around 2 seconds to pre-fill the form.
  • A game can be edited via either the collection page or its detailed information page by clicking on the edit icon.
  • A game can be deleted via either the collection page or its detailed information page by clicking on the delete icon. It will permanently remove the game and all its associated data from the collection.
